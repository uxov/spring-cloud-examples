## Build And Run
### Requirements 
- JDK 15  
- Gradle 
 
### Build
Such as the following commands:
```shell
cd spring-cloud-examples/spring-cloud-eureka
cd eureka-server
gradle build 
```

### Run
```shell
gradle bootRun
```
